# QE Director Agent

MVP workspace for a quality engineering director operating system.

## Quick start
```bash
./scripts/help
./scripts/bootstrap
./scripts/makeenv
# edit .env and set your real KIMI_API_KEY
./scripts/doctor
./scripts/skills
./scripts/newcase quality_gate my_case
./scripts/runquick quality_gate my_case
```
