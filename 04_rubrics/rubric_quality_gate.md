# Rubric Quality Gate

## Decision options
- Go
- Conditional Go
- No Go

## Go
Use this when:
- No blocker issue remains
- Regression coverage is sufficient
- Monitoring is ready
- Rollback is feasible
- Key risks are known and accepted

## Conditional Go
Use this when:
- There is no blocker, but some moderate risks remain
- Compensating actions are clearly defined
- Monitoring and rollback are ready
- Owners and timelines are explicit

## No Go
Use this when:
- Blocker issues remain unresolved
- Regression is not sufficient
- Monitoring is missing for critical areas
- Rollback is unclear or not feasible
- Release risk is not understood

## Required output
- Decision
- Top risks
- Required actions
- Priority
- Director reminder
