# Skill 06 Management Reporting

## Purpose
Generate director-level weekly or monthly reporting outputs for quality engineering management.

## Input
- Reporting period
- Key progress
- Core metrics
- Key problems
- Risks
- Actions completed
- Next-step plan
- Audience
- Expected output format

## Output
- Conclusion
- Key points
- Analysis
- Recommended actions
- Priority
- Director reminder

## Core reporting points
- What changed in this period?
- What matters most to management?
- Which risks need escalation?
- Which actions are in progress?
- What should happen next?
